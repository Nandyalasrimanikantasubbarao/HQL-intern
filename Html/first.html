Block level elements

Block-level elements in HTML are those that start on a new line in the document layout and stretch out as wide as their parent container, effectively creating a "block." Examples of block-level elements include <div>,<p>,<h1> to <h6>, <ul>,<ol>,<li>,<table>, and <form>.

Inlin e Elements

Inline elements in HTML are those that do not start on a new line and only take up as much width as necessary.Examples of inline elements include <a>, <span>, <img>, <strong>, and <em></em>,<b>
